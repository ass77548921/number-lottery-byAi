# 尾牙號碼抽獎

單頁式網頁抽獎工具，適用於尾牙、活動現場依號碼抽獎，**不重複**抽出指定範圍內的號碼，並具備滾動動畫與煙火揭曉效果。

---

## 功能特色

- **自訂號碼範圍**：輸入最小與最大號碼（如 1～100），決定抽獎號碼池
- **一次一號**：每次點擊「抽獎」只抽出一個號碼，方便逐位揭曉
- **不重複抽獎**：同一輪內已抽過的號碼不會再出現，直到按下「重置」
- **抽獎動畫**：點擊後會先快速滾動數字，再由快到慢揭曉當前號碼
- **煙火特效**：揭曉瞬間以多發煙火從中央向外擴散（純 CSS）
- **已抽列表**：下方依序顯示本輪已抽出的號碼
- **重置**：清空已抽列表並還原號碼池，可重新設定範圍後再抽
- **RWD**：支援桌面、平板、手機，含安全區域與橫向手持優化

---

## 技術說明

- **純前端**：HTML5、CSS3、原生 JavaScript，無需後端與資料庫
- **無額外套件**：未使用任何第三方 JS 函式庫或 CDN
- **檔案結構**：
  - `index.html` — 頁面結構
  - `style.css` — 樣式與 RWD、動畫
  - `script.js` — 號碼池邏輯、抽獎與動畫觸發

---

## 使用方式

1. 用瀏覽器直接開啟 `index.html`，或透過本地伺服器開啟：
   ```bash
   # Python 3
   python -m http.server 8000

   # Node.js（若已安裝 npx）
   npx serve .
   ```
2. 在「最小號碼」「最大號碼」輸入範圍（例如 1 與 100）。
3. 點擊「抽獎」：會先播放滾動動畫，再揭曉一個號碼並顯示煙火。
4. 重複點擊可繼續抽下一個號碼；抽完後按鈕會顯示「已全部抽完」。
5. 點擊「重置」可清空結果並依目前範圍重新開始。

## 響應式設計（RWD）

| 斷點 | 適用裝置 | 說明 |
|------|----------|------|
| ≤ 1024px | 平板 / 小桌面 | 容器與字級略縮、維持橫向排版 |
| ≤ 768px | 平板直向 / 大手機 | 範圍區改直向、按鈕全寬、觸控友善 |
| ≤ 480px | 一般手機 | 再縮小字體與間距、按鈕至少 44px 高 |
| ≤ 360px | 小螢幕手機 | 最小化留白與字級 |
| 高度 ≤ 500px 橫向 | 手機橫向 | 減少垂直間距、避免內容被裁切 |

並支援 `safe-area-inset`，在有劉海或圓角的裝置上可正常顯示。
